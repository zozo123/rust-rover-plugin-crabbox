The promise

Not a shell launcher. A remote proof loop.

Crabbox is the control plane: lease or delegate a sandbox, sync the checkout, run the command, stream output, retain evidence, and clean up. The plugin is the RustRover cockpit for that loop.

01

Local trust

Keep code, inspections, and edits in RustRover. No remote IDE migration.

02

Remote capacity

Run `cargo test`, `cargo clippy`, or smoke commands on Crabbox-backed sandboxes.

03

Returned proof

See streamed output, clickable run ids, lease ids, logs, and artifacts from the IDE.

End-to-end flow

From blank install to remote Rust test.

  1. Configure. Paste the Islo API key into Settings > Crabbox.
  2. Preflight. Run Tools > Crabbox > Doctor Islo.
  3. Smoke. Run Tools > Crabbox > Run Islo Rust Smoke.
  4. Prove. Run Tools > Crabbox > Run Cargo Test on Islo.

Install

Test the plugin from GitHub.

Download the release zip, then in RustRover open Settings > Plugins > gear menu > Install Plugin from Disk...
